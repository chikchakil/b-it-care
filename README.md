# B-IT CARE

B-IT CARE is a business website for IT management, cloud services, backups,
cybersecurity, servers, and professional support for businesses.

## Tech Stack

This project is based on Next.js and Tailwind CSS.

## Installation

```bash
npm install
```

## Local Development

```bash
npm run dev
```

Open the local site at:

```text
http://localhost:3000
```

## Environment Variables

There is currently no need for an `.env` file.
