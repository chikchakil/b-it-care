# Project Context

## Project Goal

B-IT CARE is a marketing website for a business IT services provider. The site
presents services such as managed IT, Microsoft 365, cybersecurity, backups,
cloud infrastructure, servers, and support.

## Current Status

- The project runs locally on macOS.
- `npm install` completes successfully.
- `npm run dev` works.
- The local site opens at `http://localhost:3000`.
- Git is connected to `origin/main`.
- No environment file is currently required.

## General Structure

- `app/` - Next.js application routes, layout, page content, and global styles.
- `components/ui/` - reusable UI components.
- `lib/` - shared utility functions.
- Root config files - Next.js, TypeScript, Tailwind, PostCSS, ESLint, npm, and shadcn configuration.

## Working Rules

- Do not commit or push without explicit approval from the project owner.
- Do not delete files without explicit approval.
- Explain before making any significant change.
